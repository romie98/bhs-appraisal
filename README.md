# Teacher Portfolio Website

A React-based teacher portfolio website built with Vite, React Router, and TailwindCSS.

## Features

- Responsive sidebar navigation
- Mobile-friendly collapsible sidebar
- Multiple pages: Home, GP1-GP6, About, Contact
- Clean, modern layout with TailwindCSS

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/
│   ├── GlobalLayout.jsx  # Main layout component
│   ├── Sidebar.jsx       # Navigation sidebar
│   └── Header.jsx        # Top header
├── pages/
│   ├── Home.jsx
│   ├── GP1.jsx - GP6.jsx
│   ├── About.jsx
│   └── Contact.jsx
├── App.jsx               # Router configuration
├── main.jsx              # Entry point
└── index.css             # TailwindCSS imports
```
